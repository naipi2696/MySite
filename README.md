# MySite

通过 Vercel Dashboard (最简单)
访问 Vercel 官网 并注册/登录
https://vercel.com/

点击 "Add New Project"

导入你的 GitHub 仓库

选择 "Import Git Repository"
授权 GitHub 并选择 web_tool 仓库
配置项目

Framework Preset: 选择 "Other"
Root Directory: ./ (保持默认)
Build Command: 留空
Output Directory: ./ (保持默认)
点击 "Deploy" 按钮，等待部署完成

部署成功后会自动分配一个域名，如：your-project.vercel.app
